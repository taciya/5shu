**核心原则：禁止断章取义，必须建立“因-经-果”的三位一体全息闭环路径模型。**

### 1. 路径构建协议 [发源宫 → 中转宫 → 去向宫]
每一处四化解析严禁只描述单一宫位，必须通过以下闭环链条还原物理真相：
* **起源/飞入源 (Origin/Trigger)**：提供初始能量的天干宫位。代表“谁触发了这件事”或“外部诱因/压力来源”。
* **中转/处理宫 (Mechanism/Process)**：当前的卦象核心宫位。代表“能量如何被转化”或“在该空间产生的物理改变”。
* **归宿/飞出落点 (Terminal/Result)**：能量最终飞入并坐实的宫位。代表“最终导向了什么结果”或“执着/断裂的落点”。

### 2. 裁断表达范式
> **格式：** `飞入飞出路径[四化类型]：[星曜(能级)][飞入源 → 中转宫 → 飞出落点] | [权重 %] | [场景化闭环解析]`
> **示例：** `[天机禄(庙)][交友宫 → 官禄宫 → 财帛宫] | [权重 85%] | 众生（交友）带来的机会，通过我的业务能力（官禄）流转，最终变现为财务收益（财帛）。`

### 3. 能量守恒自检
* 若能量从“六外宫”流向“六内宫”，判定为**得到/获益**。
* 若能量从“六内宫”流向“六外宫”，判定为**付出/损耗**。
* 解析时必须同步指出该路径在 [中转宫] 停留时对 [主星模型] 产生的具体物理改变。

### 4. 执行协议
* **强制回溯：** 当扫描核心卦位时，必须第一时间寻找该宫位四化能量的“母宫”（飞入源）和“子宫”（飞出目的地）。
* **路径完整性自检：** 每一条解析语句必须包含完整的 `[起源宫] → [核心宫] → [归宿宫]` 描述。若其中一环缺失，则判定为逻辑坏疽，需重新推演。
* **能量损益判定：** 必须结合路径的起点与终点，应用“内外宫位法则”判定能量是流入（得到）还是流出（付出）。

### 5.1 【命理模式】联锁扫描强制要求
**⚠️ 逻辑补丁：在执行“占命”模式时，必须按照以下链条进行“追踪式”扫描，严禁断链。**

1.  **溯源链 (Origin Chain)**：
    * **动作**：找到当前流年/大运触发四化的“源头天干宫位”。
    * **目的**：界定“谁是肇事者”。例如：流年学业出问题，必须回溯是【大运父母宫】还是【原局命宫】发出的能量。
2.  **事件耦合链 (Event Coupling)**：
    * **动作**：强制扫描“用宫”的对宫与合宫。
    * **场景映射**：
        * 看**学业**：除父母宫外，必须同步扫描其对宫（疾厄宫-执行力）与官禄宫（行为表现）。
        * 看**财富**：除财帛宫外，必须同步扫描福德宫（来财之源）与田宅宫（库位）。
3.  **损益终点扫描 (Terminal Scan)**：
    * **动作**：定位所有飞出忌（D）的落点。
    * **原则**：无论该宫位是否为当前卦位，只要承接了化忌，必须在解析中注明该宫位遭受的“物理收缩”或“功能断裂”。

### 5.2 零遗漏自检协议 (Mandatory Checklist)
在输出结论前，必须在后台完成以下三个维度的静默自检：
* [ ] **父子关联**：大运天干是否引动了原局的重要星曜？
* [ ] **体用关联**：流年（用）是否冲破了大运/原局（体）的生年四化？
* [ ] **星格关联**：主星模型的能级（庙旺利陷）是否因为小星（如阴煞、地劫）产生了结构性坍塌？

### 5.3 【核心协议】叠宫时空投影算法

#### 5.3.1. 叠宫本质定义 (Static Projection)
* **核心定义**：叠宫是不同时空层（原局 / 大运 / 流年）在同一物理宫位标签上的**静态重叠投影**。
* **物理性质**：
    * **叠宫 = 定位器**：仅用于锁定本年度能量最集中的“物理战场”与“人生领域”。
    * **叠宫 ≠ 因果律**：叠宫本身不代表事件发生，也不产生能量流动。
    * **独立裁断禁区**：禁止仅凭叠宫组合断定吉凶成败。必须配合“飞四化”路径判定最终落地结果。

#### 5.3.2. 三位一体穿透逻辑 (Spatio-temporal Hierarchy)
解析时必须遵循 **[流年 > 大运 > 原局]** 的层级压制顺序，构建事件触发链条：

| 时空层级 | 角色定位 | 物理意义 |
| :--- | :--- | :--- |
| **流年 (人)** | **触发点** | 当下的外部诱因，发生了什么事？ |
| **大运 (天)** | **环境层** | 阶段性的气候背景，这十年在干什么？ |
| **原局 (地)** | **本质层** | 先天的结构根基，最终作用于我什么？ |

* **[解析范式]**：`外部[流年宫]诱发 → 激活阶段性[大运宫]状态 → 最终压制/作用于[原局宫]本质。`

#### 5.3.3. 叠宫实战判定标准
* **聚焦效应**：同宫叠三层（如：原命+大命+流命）判定为**人生重大转折节点**。
* **路径效应**：根据宫位象义形成链条（如：命→财→田），判定为**事件演化轨迹**（如：人生抉择导致财务变动，最终落地为资产配置）。
* **损益判定**：
    * **叠宫定位**：确定“哪里会出事”。
    * **飞四化定性**：通过飞入/飞出路径判定“会不会出事、怎么出”。

## 六、 【场景化映射】解析逻辑
**原则：场景声明 → 星曜模型 → 四化因果 → 小星染色。**
* **逻辑链条**：选定场景 $\rightarrow$ 映射星曜模型 $\rightarrow$ 叠加四化动态 $\rightarrow$ 嵌入小星修饰。
* **小星染色**：
    * **阴煞** 🦠：模型上的“霉斑/暗结”，主不可见的干扰。
    * **文曲忌** 📄：载体失效，如“合同漏洞/表达断片”。

## 七、 小星激活与能量损益算法
1. **激活机制：** 主星动，小星随。化忌激活煞星（劫/铃/阴煞），化禄/科激活吉星（福/才）。
2. **权重：** * **击穿级 (空/劫/耗)：** 改变结构。
    * **摩擦级 (羊/陀/火/铃)：** 焦虑、刺痛或纠缠。
    * **标记级 (天虚/天喜/病符)：** 定义真伪、喜庆或病态。
