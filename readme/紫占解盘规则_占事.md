# 解盘总规则

## 一、基础框架
- 使用三层卦象：主卦 / 十分卦 / 分钟卦
- 每一层卦都必须同时解读：
  - 本宫星曜结构
  - 向心四化 / 离心自化
  -  飞四化（飞出） = 发生机制层
    - 飞禄：资源被送去哪里运作
    - 飞权：推动 / 施压 / 调用去哪里
    - 飞忌：问题 / 压力被抛给哪里
  - 追四化（飞入） = 溯源定位层
    - 追禄：好处源头在哪里/为了什么
    - 追权：谁在真正推动/施压方/调用方
    - 追忌：问题根源/压力源/症结点在哪里
    - 追科：名义 / 合理性来源
  - 裁断结论 = 现实解释层
  - 宫内小星的解读，如下表格，最好做四化前后的对比解读，让我对星性更加了解：
    - | 结构   | 不化忌   | 化忌    |
    - | ---- | ----- | ----- |
    - | 太阳   | 外参可信  | 外参降权  |
    - | 铃星   | 噪声    | 被误读信号 |
    - | 天虚   | 可忽略空位 | 参照失焦  |
    - | 天才   | 正向自适应 | 反复修正  |
    - | 阴煞   | 可补偿扰动 | 持续推力  |
    - | 整体表现 | 稳中有抖  | 抖中回收  |

## 二、四化定义统一
裁断结论 = 现实解释层
- 飞四化：本宫【飞出】四化能量，代表该宫主动输出、外放、消耗或施压
- 追四化：四化能量【飞入】本宫，代表该宫被动承接、被调用、被牵引
- 四化联动：裁断结论时，需要给出裁断逻辑。比类似以下内容：
  - ① 飞禄（发生机制）
    - 资源被往外送
    - 不在交友宫直接结算
  - ② 子女宫（接口）
    - 不出结果
    - 只跑生成 / 执行 / 中间步骤
  - ③ 天机（核心运作方式）
    - 用规则
    - 用流程
    - 用“不是我能决定”的逻辑
  - 👉 合并裁断（不是一句话跳出来的）
    - 标准紫占裁断句应该是：
    - 这笔钱的“生机”
    - 被从“人际协调（交友宫）”
    - 转移到“流程运作（子女宫天机）”，
    - 因此呈现为——
    - 不是不给，而是一直在系统里跑。

## 三、双宫同卦原则
- 仅存在两组（同宫干）：
  - 子宫 + 寅宫
  - 丑宫 + 卯宫
- 一旦构成双宫同卦：
  - 四化必须合并判断
  - 不可将单宫飞忌直接判为崩坏

## 四、误判防火墙规则
- 必须检查：
  - 忌来自何宫（内/外）
  - 是否存在转存路径
  - 是否有制度星曜托底

## 八、教学级解盘原则
- 所有结论必须能回溯到：
  星曜根本象义 → 四化机制 → 宫位接口功能
- 禁止黑箱结论
