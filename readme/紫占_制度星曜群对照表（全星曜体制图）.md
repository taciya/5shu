以下为【制度星曜群】的完整体制化对照表。本表不是性格或吉凶表，而是“当一个社会/组织/修行体系要长期运作时，各星曜在制度中的必然分工”。

——————————
一、制度生成核心链（治理中枢）
——————————

紫微
→ 体系主权 / 最终裁量权
→ 决定“这套系统要不要存在、往哪里走”
→ 皇权、最高决策、宗主、法统源头

天府
→ 资源与秩序中枢
→ 决定“这套系统靠什么活、能撑多久”
→ 财政、库藏、后勤、组织稳定器

天相
→ 合法化与程序接口
→ 决定“是否符合规范、是否可复制与承接”
→ 制度、规则、资格认证、流程正当性

天梁
→ 修复与延命系统
→ 决定“出问题后系统还能不能继续活”
→ 监察、道义、善后、保护伞

——————————
二、制度冲突与边界系统（问题生成层）
——————————

巨门
→ 定义权争夺机制
→ 触发“这件事到底算不算数”
→ 争议、质疑、申诉、言词对抗

廉贞
→ 边界测试与灰区探测
→ 触发“有没有越线、有没有利益纠缠”
→ 纪律、欲望、权色、违规试探

七杀
→ 极端决断机制
→ 触发“不经程序、直接解决”
→ 军事、强制、非常状态

破军
→ 旧制度拆解器
→ 触发“这套制度已经不适用了”
→ 革命、重组、推翻、清盘

——————————
三、制度运转与执行系统（现实操作层）
——————————

武曲
→ 投入—产出转化器
→ 决定“制度是否有效率、有没有回报”
→ 财务、执行、绩效、实干部门

太阳
→ 外部显化与责任承担
→ 决定“制度对外怎么看、谁扛责任”
→ 公共形象、领导露出、对外承诺

太阴
→ 内部滋养与隐性支持
→ 决定“制度内部是否稳定、成员是否被照顾”
→ 福利、家庭系统、隐性资源

——————————
四、制度智能与调整系统（认知与策略层）
——————————

天机
→ 演算与调整中枢
→ 决定“制度如何应变、如何升级”
→ 策略、技术、规划、智囊

贪狼
→ 扩张与欲望引擎
→ 决定“制度是否要扩张、吸引人”
→ 市场、关系、诱因、增长动能

天同
→ 缓冲与人性接口
→ 决定“制度是否太硬、是否需要休息”
→ 福利缓冲、妥协、延迟

——————————
五、辅助星曜在制度中的角色定位（简表）
——————————

左辅 / 右弼
→ 制度协助与执行辅助
→ 幕僚、助手、配套力量

文昌 / 文曲
→ 制度文本与话语系统
→ 文件、法条、教材、传播

禄存
→ 制度黏着资源
→ 固定利益、既得结构

擎羊 / 陀罗
→ 制度摩擦与代价
→ 阻力、牺牲、磨损

火星 / 铃星
→ 制度不稳定因子
→ 突发、失控、爆点

地空 / 地劫
→ 制度空转与流失
→ 名存实亡、资源断裂

——————————
【一句总纲（教学用）】

巨门让制度不得不面对问题，
廉贞试探边界，
天相决定能否合法，
紫微裁量方向，
天梁保证系统不死，
破军与七杀负责非常时刻，
武曲与天府让制度真的能活下去。
