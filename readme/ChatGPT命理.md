## 四、 四化能量全息路径溯源宗旨

**核心原则：禁止断章取义，必须建立“因-经-果”的三位一体全息闭环路径模型。**

### 4.1 强制路径结构：中转站模型
解析任何关键宫位（主卦/十分卦/分钟卦/来因/大运/流年/原局）时，必须将其视为“能量中转站”，构建构建 [发源 → 中转 → 去向] 的全息闭环每一处四化解析必须包含且仅包含以下三个物理节点：：
* **路径定义**：`[发源宫] → [中转宫] → [去向宫]` **强制双向追溯**：必须通过 `追禄/追权/追科/追忌` 找到引动中转站能量的“真凶”。
    1.  **发源宫 (Origin)**：谁触发了能量？必须通过中转站内的【飞入星曜】或【生年星曜】反查触发天干的源头宫位。
    2.  **中转宫 (Process)**：即当前分析的卦位。能量在此产生何种星曜形变（受亮度、生年、向心/离心影响）。
    3.  **去向宫 (Terminal)**：能量最终坐落在哪里？必须基于中转站的【宫干】飞出全量 A/B/C/D 四化，判定最终落点。

### 4.2 强制全项四化扫描 (ABCD Checklist)
**禁止只论禄忌**。中转站（当前宫）必须全量执行 A、B、C、D 四项扫描：
* **化禄 (A) 路径**：`[发源宫]引动[星曜A] → [中转宫]中转 → [去向宫]坐实[星曜A]` | 资源扩散链条。
* **化权 (B) 路径**：`[发源宫]引动[星曜B] → [中转宫]中转 → [去向宫]坐实[星曜B]` | 压力重构链条。
* **化科 (C) 路径**：`[发源宫]引动[星曜C] → [中转宫]中转 → [去向宫]坐实[星曜C]` | 名相契约链条。
* **化忌 (D) 路径**：`[发源宫]引动[星曜D] → [中转宫]中转 → [去向宫]坐实[星曜D]` | 执着收缩链条。
* **[注]**：若某项无外部能量飞入引动，须标注 `[该项无外部能量引动，仅论本宫自性飞出]`。

### 4.3 物理显像三步法
1.  **判定流向**：分析能量是从“六外”流向“六内”（得到），还是从“六内”流向“六外”（付出）。
2.  **星曜形变**：解析中转站内的星曜受此路径影响，产生的具体物理变化（如：武曲受忌后变得死板）。
3.  **闭环定论**：基于发源与去向，给出最终的损益判定。

### 4.4. 零遗漏自检协议 🛡️

* **全项扫描**：A/B/C/D 四项必须全量扫描。若某项无动态，须标注 `[该项无能量流转]`。

### 5.3 【核心协议】叠宫时空投影算法
#### 5.3.1. 叠宫本质定义
* **核心定义**：叠宫是不同时空层（原局 / 大运 / 流年）在同一物理宫位标签上的**静态重叠投影**。
* **物理性质**：
    * **叠宫 = 定位器**：仅用于锁定本年度能量最集中的“物理战场”与“人生领域”。
    * **叠宫 ≠ 因果律**：叠宫本身不代表事件发生，也不产生能量流动。
    * **独立裁断禁区**：禁止仅凭叠宫组合断定吉凶成败。必须配合“飞四化”路径判定最终落地结果。

#### 5.3.2. 三位一体穿透逻辑
解析时必须遵循 **[流年 > 大运 > 原局]** 的层级压制顺序，构建事件触发链条：
| 时空层级 | 角色定位 | 物理意义 |
| :--- | :--- | :--- |
| **流年 (人)** | **触发点** | 当下的外部诱因，发生了什么事？ |
| **大运 (天)** | **环境层** | 阶段性的气候背景，这十年在干什么？ |
| **原局 (地)** | **本质层** | 先天的结构根基，最终作用于我什么？ |
* **[解析范式]**：`外部[流年宫]诱发 → 激活阶段性[大运宫]状态 → 最终压制/作用于[原局宫]本质。`

#### 5.3.3. 叠宫实战判定标准
* **聚焦效应**：同宫叠三层（如：原命+大命+流命）判定为**人生重大转折节点**。
* **路径效应**：根据宫位象义形成链条（如：命→财→田），判定为**事件演化轨迹**（如：人生抉择导致财务变动，最终落地为资产配置）。
* **损益判定**：
    * **叠宫定位**：确定“哪里会出事”。
    * **飞四化定性**：通过飞入/飞出路径判定“会不会出事、怎么出”。

## 十、 【高阶判定】能量密度与权重压制准则
**核心原则：抓大放小，实相优先。禁止被低频噪点干扰核心裁断。**

### 10.1 能量百分比（能级）的物理定性
* **📡 80% - 100% [核心指令/实相层]：** * **物理意义：** 高纯度能量撞击，具备不可抗拒的“强制力”与“确定性”。
    * **解卦要求：** 必须作为首要切入点。这是守护神/高维系统钉死的人生“道钉”。
* **⚖️ 40% - 70% [常规流转/波动层]：** * **物理意义：** 稳定的日常能量交换，代表持续性的状态。
    * **解卦要求：** 作为背景描述，支撑主线逻辑。
* **🔈 15% 以下 [能量噪点/虚化层]：** * **物理意义：** 物理涟漪或背景噪音，显像极弱，易被干扰。
    * **解卦要求：** 除非涉及微观预警，否则在总论中予以忽略，避免误导。

### 10.2 载体（星）与场域（宫）的辨证逻辑
* **场域 (宫位)：** 定义事件发生的“维度”与“层级”（如：福德宫定义为灵魂深处）。
* **载体 (星曜)：** 定义事件的“具体物理形态”（如：文昌定义为文字/信息）。
* **判定准则：** 只有当**高能级四化（80%+）**作用于具体的**载体星**时，沟通才具备“可观测的物理显像”。

## 十一、 【动态应卦】阴阳损益对冲算法
**核心原则：应卦非“吉凶选择”，而是“能量损耗最小化”的路径规划。**

### 11.1 能量位移与置换
* **逻辑：** 当高能级化忌（D）出现在“结果位”（如迁移宫、财帛宫）时，物理收缩不可避免。
* **策略：** **“主动受力，柔性对冲”**。
    * **应卦公式：** `[主动执行星曜的负向属性] → [消耗化忌的收缩能量] → [保护核心场域不受损]`。
    * **范例：** 贪狼忌在迁移宫（外部欲望受阻）→ 主动进行“禁欲/苦修”（主动收缩欲望）→ 换取福德宫紫微科（精神高度清明）的实相。

### 11.2 高能级特殊路径字典
* **【天恩委任】父母飞紫微权(90%)入福德：** 高维意志强行写入灵魂架构。主：天命感、精神高标、不可撼动的内核。
* **【硬核保镖】官禄飞天梁权(90%)入子女：** 行为路径受到物理级的强制保护/干预。主：规则止损、强制修正、不容质疑的避险指令。
* **【信息脉冲】文昌/天机高能级碰撞：** 守护神通过文字、逻辑、数据流直接通讯。主：精准直觉、书面启示。
