
# 紫占解盘规则总导出（当前版本）

## 一、基础框架
- 使用三层卦象：主卦 / 十分卦 / 分钟卦
- 每一层卦都必须同时解读：
  - 本宫星曜结构
  - 向心四化 / 离心自化
  - 飞四化（飞出）
  - 追四化（飞入）

## 二、四化定义统一
- 追四化：四化能量【飞入】某宫，代表该宫被动承接、被调用、被牵引
- 飞四化：四化能量【飞出】某宫，代表该宫主动输出、外放、消耗或施压
- 四化联动：某宫飞出四化 → 改变其他宫位状态，再反向影响本宫

## 三、三层卦统一规则
- 主卦：问题的“结构根因”
- 十分卦：问题如何在现实中展开、卡住或失衡
- 分钟卦：当事人主观感受与即时反应机制
- 解盘顺序必须是：主卦 → 十分卦 → 分钟卦（不可倒置）

## 四、双宫同卦原则（紫占专用）
- 仅存在两组：
  - 子宫 + 寅宫（同宫干）→ 算力 / 认知处理系统
  - 丑宫 + 卯宫（同宫干）→ 制度 / 承接 / 规范系统
- 一旦构成双宫同卦：
  - 四化必须合并判断
  - 不可将单宫飞忌直接判为崩坏

## 五、天机专项判断规则
- 天机 = 运算 / 推演 / 认知流动
- 天机飞四化 ≠ 必然算力拆散
- 区分三种状态：
  1. 算力崩坏型（内忌 + 无承接）
  2. 停机型天机（外忌 + 双宫同卦 + 有托底）
  3. 转存型天机（制度星曜介入，算力转后台）

## 六、制度星曜介入条件
- 可稳定天机的星曜：
  - 天相（合法性 / 流程 / 可检验）
  - 天府（库存 / 承载 / 缓冲）
  - 天梁（上位规则 / 保护性暂停）
- 判断方式：
  - 是否在双宫同卦系统内
  - 是否以科 / 权形式介入
  - 是否形成长期承接而非短期解救

## 七、误判防火墙规则
- 不得因：
  - 反应慢
  - 停机
  - 不输出
  直接判定为“笨”
- 必须检查：
  - 忌来自何宫（内/外）
  - 是否存在转存路径
  - 是否有制度星曜托底

## 八、教学级解盘原则
- 所有结论必须能回溯到：
  星曜根本象义 → 四化机制 → 宫位接口功能
- 禁止黑箱结论
