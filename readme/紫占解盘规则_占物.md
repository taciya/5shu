# 解盘总规则

## 一、基础框架
- 使用三层卦象：主卦 / 十分卦 / 分钟卦
- 每一层卦都必须同时解读：
  - 本宫星曜结构
  - 向心四化 / 离心自化
  - 飞四化（飞出）= 发生机制层
  - 追四化（飞入）= 溯源定位层
  - 裁断结论 = 现实解释层
  
## 二、四化定义统一
裁断结论 = 现实解释层
- 飞四化：本宫【飞出】四化能量，代表该宫主动输出、外放、消耗或施压
- 追四化：四化能量【飞入】本宫，代表该宫被动承接、被调用、被牵引
- 四化联动：裁断结论时，需要给出【飞出】与【飞入】四化能量之间的互动关系

## 三、双宫同卦原则（紫占专用）
- 仅存在两组：
  - 子宫 + 寅宫（同宫干）→ 算力 / 认知处理系统
  - 丑宫 + 卯宫（同宫干）→ 制度 / 承接 / 规范系统
- 一旦构成双宫同卦：
  - 四化必须合并判断
  - 不可将单宫飞忌直接判为崩坏

## 四、误判防火墙规则
- 必须检查：
  - 忌来自何宫（内/外）
  - 是否存在转存路径
  - 是否有制度星曜托底

## 八、教学级解盘原则
- 所有结论必须能回溯到：
  星曜根本象义 → 四化机制 → 宫位接口功能
- 禁止黑箱结论
